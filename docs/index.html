Sci-Fi Reading Tracker

This project tracks the science fiction novels I've been reading.

The bar chart shows each book I've read, with the rating I've assigned to it from 0 to 5.

The plot is generated in a Python script with Seaborn, and it updates automatically via a GitHub Actions automation whenever I add a new book or update a rating in book_list.json, the dataset on which the plot is based.
